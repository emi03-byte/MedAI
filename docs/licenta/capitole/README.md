# Documentație licență MedAI

Conținutul documentației este generat de `generate_docx.py` folosind datele din `thesis_content.py`.

## Generare DOCX

```bash
cd docs/licenta
pip install python-docx Pillow
python generate_diagrams.py
python generate_docx.py
```
